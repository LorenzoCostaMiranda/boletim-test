# RMarkdown

### Abreviações

|  RMarkdown   | LaTeX Output |
| :----------: | :----------: |
| `[abbr:key]` | `\abbr{key}` |

### Referenciamento

|   RMarkdown    |    LaTeX Output    |
| :------------: | :----------------: |
|  `[fig:key]`   |  `\ref{fig:key}`   |
|  `[tab:key]`   |  `\ref{tab:key}`   |
|  `[box:key]`   |    `\ref{key}`     |
|   `[eq:key]`   |   `\ref{eq:key}`   |
|   `[ch:key]`   |   `\ref{ch:key}`   |
|  `[sec:key]`   |  `\ref{sec:key}`   |
| `[subsec:key]` | `\ref{subsec:key}` |
